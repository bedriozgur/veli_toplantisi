# Veli Toplantısı

Vite + React tabanlı veli toplantısı takip uygulaması.

Özellikler:
- Öğretmenler sekmesi
- Sınıflar sekmesi
- Öğrenciler ve QR akışı
- Her öğrenci için kişiselleştirilmiş veli görünümü
- Görüşme tamamlandı işaretleme
- Not alma ve e-posta ile özet gönderme

Kurulum:
```bash
npm install
npm run dev
```

Build:
```bash
npm run build
```
